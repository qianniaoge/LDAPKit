## 简介

自用的`LDAP`工具（无高级用法仅支持简单的漏洞复现）

无需自行编译`Java`文件和开启`HTTP`服务，一切自动

只要一行：`java -jar LDAPKit.jar [命令]`

例如：`java -jar LDAPKit.jar notepad.exe`
